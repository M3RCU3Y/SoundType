# Privacy

SoundType is designed as a local-only keyboard sound utility.

It does not record what you type. It does not save typed words, typed characters, or key history. It does not send keystrokes anywhere.

SoundType listens for keyboard events only so it can play a sound when a key is pressed. Settings may store key names used for configuration, such as excluded keys, and app process names used for app-specific rules.

No account is required.
No cloud service is required.
No telemetry is enabled by default.
