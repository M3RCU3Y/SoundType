# Reference Notes

SoundType's WPF theme polish uses Keyboard Sounds Pro as a conceptual visual reference. The adapted ideas are limited to theme-level treatment: dark elevated cards, subtle borders and shadows, emerald accent states, and distinct disabled/error/warning colors. No React, CSS, Go, or other source implementation was copied into SoundType.

Reference reviewed:

- Keyboard Sounds Pro, MIT License, copyright 2025 Nathan Fiscaletti.
- Mechvibes, MIT License, copyright 2021 Hai Nguyen. This branch did not adapt Mechvibes implementation details.

SoundType keeps its existing WPF architecture and control flow. The styling changes live in WPF resource dictionaries and templates so buttons, text boxes, combo boxes, list boxes, sliders, tabs, check boxes, and group boxes remain native WPF controls.
